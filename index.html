1. História da Web

Em 1944 surgiu o primeiro computador, chamado de MARK I. Um computador muito grande, eletro mecânico, baseado em valvulas. Em 1946 surgiu o Colossus que foi criado na Alemanha. E no mesmo ano surgiu o ENIAC nos EUA. Essa foi a primeira geração de computadores. O termo BUG* surgiu nessa época por Grace Murray Hopper, umas das primeiras programadoras do MARK I.

A segunda geração de computadores aconteceu de 1959 a 1965. Essa geração foi marcada pela troca das valvulas pelos transistores, deixando os computadores menores.

A terceira geração de computadores que foi de 1965 a 1970 foi marcada pela chegada dos circuitos integrados (Transistor, Capacitores, Resistores). Que acabaram deixando os computadores ainda menores.

A quarta geração começa em 1971 e vem até os dias de hoje. Os circuitos integrados foram trocados por microprocessadores, os chips continuaram sendo utilizados. E foi a partir desse momento que começaram a surgir os computadores pessoais, os Smartphones e muitas outras tecnologias.

Computadores quânticos, o Meta Verso, Cryptomoedas, Inteligência artificial, podem ser a quinta geração de computadores.

A primeira conexão estabelicida entre dois pontos foi feita em 29/10/1969 pela ARPANET. A comunicação foi feita entre a UCLA e SRI.

Uma curiosidade é que nessa conexão foi enviada uma palavra de um ponto ao outro, a palavra "LOGIN". E no destino só chegou "LO". Então nesse mesmo dia aconteceu também a primeira queda de conexão.

Vinton Cerf e Robert Kahn criaram e implantaram o TCP/IP na ARPANET deixando a rede mais robusta. E em 1989, Tim Bernes-Lee viu a oportunidade de unir hipertexto com TCP/IP e criou a World Wide Web(WWW). E no dia 06/08/1991 foi publicada a primeira página Web. Disponivel no link: http://info.cern.ch/ .

3. Diferença entre "Aplicações Web" e "Aplicativos"

Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação. Já os Aplicativos são programas instalados no dispositivo, são desenvolvidos para uma plataforma específica ou tipo de dispositivo. O usuário deve instalar a versão de software adequada em seus dispositivos de escolha.

4. O que são Servers

Um servidor é um computador equipado com um ou mais processadores, bancos de memória, portas de comunicação, softwares e, ocasionalmente, algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD. A principal função de um servidor web é permitir que usuários acessem aplicações, serviços e dados hospedados através da internet. Ele atua como um intermediário entre um equipamento cliente, que geralmente usa um software como um navegador web, e o servidor que armazena o conteúdo desejado.

6. Tags HTML

Segue a baixo alguns exemplos de Tags do HTML:

  • i : Deixa o texto Itálico.
  • strong : Deixa o texto em Negrito.
  • Input : Cria um campo para digitação na tela.
  • img : Adiciona uma imagem.
  • h1, h2, h3, h4, h5, h6 : Usado para títulos e subtítulos.
  • p : Adiciona um parágrafo.
  • blockquote : Para criar uma citação, ou observação.
  • u : Para deixar o texto sublinhado.
  • mark : Marcador.
  • sup : Para inserir grau, potencia, asteristico, na parte de cima da linha.
  • ol : Cria uma lista ordenada.
  • ul : Cria uma lista não ordenada.
  • a : Cria um link.
  • hr : cria uma linha divisória.
  • small : Cria palavras menores.
